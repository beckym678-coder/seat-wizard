Seat Wizard User Manual

Welcome to Seat Wizard — your classroom seating chart automation tool for Google Workspace. Click any section below to expand.

🎓 1. Import Google Classroom Rosters ▶
  • Go to Seat Wizard → Import/Sync Google Classroom Rosters.
  • Select rosters to import from Google Classroom.
  • Click Import — each course becomes a “Class - [Name]” worksheet here.
  • Seat numbers are auto-assigned initially but can be randomized later.
  • When your rosters change, importing again syncs data without overwriting existing preferences.
🪄 2. Generate Seating Charts ▶
  • Go to Seat Wizard → Generate Seating Charts.
  • Select a Class and a Slide Template.
  • Click Generate Chart for one class or Generate Charts for All Classes to create a full presentation.
  • The result is a Google Slides file with one or more classroom charts.
💡 Tip: Templates must live in your Drive folder named “Seat Wizard” and begin with “Layout -”.
🎲 3. Randomize Seats ▶
  • Click Randomize Selected Class to shuffle seats for one class.
  • Or click Randomize All Classes to reshuffle every roster.
  • Seat Wizard maintains “Keep Away” and “Preferential Seat” rules during randomization.
⭐ 4. Preferential & Keep-Away Settings ▶
  • In each class sheet, column D (“Preferential Seating”) accepts Y for students who need specific seats.
  • Columns E and F (“Keep Away 1” and “Keep Away 2”) let you avoid seating conflicts.
  • Set which seat numbers are “preferential” via Seat Wizard → Set Preferential Seats.
📋 5. Layout Templates ▶
  • Layouts determine classroom structure and must be in the Seat Wizard folder in your Drive.
  • Slide templates (Google Slides) can show Student or Teacher perspectives — Seat Wizard automatically fills seat numbers with names.
  • You can create desk layouts directly in the slide or in the slide’s theme editor.
  • Labeled items are left untouched; seat numbers are replaced with student names.
  • Each desk must have a text box containing its seat number.
  • Seat text boxes retain your formatting.
  • “Keep away” rules depend on numeric adjacency — if seats are far apart numerically, separation may not apply.
  • Higher-numbered desks are filled last and left empty if there are too few students.
🧰 6. Troubleshooting ▶
  • If a chart is blank, ensure seat numbers match the numbers in your slide template.
  • Templates must be editable by you and located in the “Seat Wizard” folder.
  • Enable the Classroom API if syncing rosters gives an error.
